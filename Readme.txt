Project Name: Packet Header Analyzer.

Description: This Java program helps to analyze different packets following different protocols like
			 TCP, ICMP and UDP. It extracts information from the packet header and displays it to the user.

How to run:  
			1) In order to compile the program type in the command "javac wirebug.java".
			2) After compiling to run the program type the command "java wirebug @filename"
			
			Note: @filename is the parameter in which you can enter the packets to be read.

Packets Attached:
			
			1) tcpjmk.pcap 
			2) udpjmk.pcap
			3) icmpjmk.pcap
